## 📖 Overview

**Robogrids** is a client-side analytics application for RPA administrators and engineers. It exclusively processes **CSV files exported from the UiPath Jobs screen**. By uploading these logs, it instantly transforms data points into a high-level, interactive dashboard.

For guidance on generating these exports, see the [UiPath Documentation](https://docs.uipath.com/orchestrator/automation-cloud/latest/user-guide/exporting-grids-in-the-background).

All data is processed entirely in your browser—**100% private, with zero server uploads**.

## ✨ Core Features

- **Executive KPI Tracking**: Instantly view Total Success Rate, Active Robots, Overall Utilization, and Critical Faults.
- **Deep Visual Analytics**: Spot bottlenecks with process analytics, completion timelines, utilization pie charts, and top faulted robot tracking.
- **Robot Gantt Timelines**: Map exact robot availability and execution states through the day.
- **100% Client-Side Processing**: Securely parse and consolidate multiple Orchestrator CSV log exports instantly via your browser.
- **Export & Documentation**: High-resolution PNG dashboard exports and in-app built-in documentation.

## 🛠️ Technology Stack

Built with **React 19**, **TypeScript**, **Vite**, **Tailwind CSS 4**, **Recharts**, and **Motion**.
